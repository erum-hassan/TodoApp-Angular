## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Install Angular 

```bash
  npm install angular@latest
```

Start the server

```bash
  ng serve --open
```
